CSS (Cascading Style Sheets) is used to control the presentation, formatting, and layout of web pages. By separating the content (HTML) from the style (CSS), web developers can maintain and update their sites more efficiently. CSS enhances the visual appeal of web pages and provides flexibility in design.

Key Concepts of CSS:

  1. Selectors: Used to select the HTML elements you want to style.
  2. Properties: Used to apply styles to the selected elements.
  3. Values: The values assigned to the properties.

Types of CSS

  1. Inline CSS: Applied directly within an HTML element using the style attribute
  2. Internal CSS: Defined within a style element inside the hea> section of an HTML document.
  3. External CSS: Linked to an HTML document using the element. This is the preferred method for larger websites.
